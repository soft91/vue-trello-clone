## 프로젝트 정보

### 프로젝트 환경

1. Vue.js (@vue/cli 4.1.1)
2. Typescript
3. Node.js Express
4. Mysql
5. Element-UI

## 시작하기

### 설치 방법

1. Clone the repo
```sh
git clone https://github.com/soft91/vue-trello-clone.git
```
2. Install NPM packages
```sh
npm install
```
